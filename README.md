# WerapolFontEndTest


## การติดตั้งโปรเจค

### 1. Clone โปรเจคจาก GitHub
เริ่มต้นด้วยการ clone โปรเจคจาก GitHub:
```bash
git clone git@github.com:werapolJa/weather.git
```
### 2. เข้าไปในโฟลเดอร์โปรเจค
cd <ชื่อโฟลเดอร์โปรเจค>

### 3. ติดตั้ง Dependencies
npm install

### 4. รันโปรเจค
npm run dev
